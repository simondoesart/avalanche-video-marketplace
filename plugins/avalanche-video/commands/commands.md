Show the user this command reference for the avalanche-video plugin, formatted cleanly:

**Skills**
- `/avalanche-video:make-video` (or just say "make a branded video") — the full pipeline: working folder → intake → strategy/hook → script workshop → continuity bible + @elements → shotlist → canonical prompts → guarded Higgsfield generation.

**Commands**
- `/avalanche-video:script` — save the current script + exact canonical prompts to the project's `script/` folder. Asks first: overwrite or new `-v{n}` iteration.
- `/avalanche-video:summary` — generate the project summary (status, hook, script synopsis, elements, generation log with credits, next steps) to `project/summary.md`. Asks first: overwrite or new iteration.
- `/avalanche-video:commands` — this list.

Note: when unambiguous, the short forms `/script`, `/summary`, `/commands` also work. Then ask if they want to run one now.
